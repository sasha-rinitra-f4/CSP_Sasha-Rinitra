HTML Basics

HTML stands for Hypertext Markup Language, it is the foundationl language for structuring content on the web. It provides the basic framework for the websites.


Tags

Tag is a piece of markup language used to indicate the beginning and end of an HTML element. There are two different types of tags: opening and closing.

Lists

Lists are a way to organize and display related items in a structured format. The different types of lists are shown in the picture above


Attributes

Attributes are additional information placed in the opening tag that gives more details to the computer about that tag. It is written inside the opening tag.


Div

In HTML, the standard markup language for documents designed to be displayed in a web browser, div is an element used to define parts of a document, so that they are identifiable when a unique classification is necessary.
